# 🛒 Mini Amazon -

A clean and responsive **Amazon UI Clone** built with HTML and CSS.

### ✨ Key Features
* 📱 **Fully Responsive** (Works on Mobile/Desktop)
* 🔍 **Sticky Navigation Bar** with Search
* 📦 **Product Grid Layout** with Hover Effects

### 🛠️ Tech Stack
* **HTML5**
* **CSS3 (Flexbox & Grid)**
